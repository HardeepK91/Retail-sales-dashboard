# Retail-sales-dashboard
Power BI Retail Sales Analysis Dashboard


Retail Sales Dashboard (2023–2026)
🔍 Overview

This project presents an interactive sales dashboard built using Power BI to analyze performance across categories, regions, and time.

🛠 Tools Used
Power BI
Excel
Power Query
DAX

📈 Key Insights
Sales increased by ~45% in 2025
Technology is the top-performing category
West region generates highest revenue
Some products consistently generate losses
